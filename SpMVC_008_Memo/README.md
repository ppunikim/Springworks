# 메모장 프로젝트
* 가장 간단하게 CRUD를 구현하기 위한 프로젝트
* mySQL, mybatis를 연동
* 한 개의 파일을 업로드 할 수 있게

## dependency
* spring-jdbc
* mybatis
* mybatis-spring
* mysql-connector
* commons-dbcp
* commons-io
* commons-fileupload
